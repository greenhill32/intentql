Agent Info

How AI agents should interact with intentql.dev

Read the contract first

Before interacting with this site, fetch and read the agent contract:

GET https://intentql.dev/agent.json

What this site provides

  • Documentation about the IntentQL specification
  • A machine-readable overview at /overview.json
  • This guidance page at /agent-info

What this site does NOT provide

  • Commerce or shopping functionality
  • User authentication
  • Write operations of any kind
  • Search endpoints
  • Contact forms or messaging

Constraints

The following constraints are declared in the contract and must be respected:

  • no_write_actions — Do not attempt POST/PUT/DELETE operations
  • no_customer_data — No user data is available
  • no_commerce — This is not a store
  • read_only — All interactions are read-only

If an intent is not listed, it is not supported

Do not invent endpoints. Do not assume capabilities. If the contract doesn't declare it, this site doesn't support it.
